правообладатель Общество с ограниченной ответственностью «Газпром нефтехим Салават».

Наименование программы для ЭВМ: «Журнал нарядов-допусков УГЭ».

Правовая основа для использования программы для ЭВМ: является правообладателем.

Краткая характеристика: программа для ЭВМ «Журнал нарядов-допусков УГЭ» предназначена для автоматизации оформления, учёта и регистрации нарядов-допусков при работе в электроустановках.

Основные функциональные возможности программы для ЭВМ:
- создание наряда из пустой формы и на основе ранее выданного;
- просмотр и печать заполненных и пустых бланков наряда;
- автоматическое заполнение даты и времени начала/окончания работ;
- редактирование и удаление наряда на этапе его подготовки;
- контроль назначения ответственных лиц с исключением недопустимых совмещений;
- отдельный учёт командированного персонала сторонних организаций;
- поэтапная регистрация работ: разрешение на подготовку, допуск бригады, завершение работ, снятие заземлений, полное окончание с блокировкой редактирования;
- автоматическое присвоение номера наряду по двум уровням регистрации с ежемесячным обнулением: 1-й уровень – диспетчерами оперативно-диспетчерского отдела, 2-й уровень – начальниками смен электроснабжения;
- отображение нарядов по уровням с удобной навигацией и календарём;
- гибкий поиск и выборка нарядов по объекту, году, присоединению, ответственным лицам, членам бригады и содержанию работ.

Область применения: процессы оформления, учёта и регистрации нарядов-допусков при производстве работ в электроустановках в деятельности Управления главного энергетика ООО «Газпром нефтехим Салават».

Достигнут полезный эффект: внедрение программы для ЭВМ позволило повысить эффективность работы подразделения УГЭ за счёт реализации:
- автоматизации оформления, учёта и регистрации нарядов-допусков;
- контроля назначения ответственных лиц с исключением недопустимых совмещений обязанностей;
- поэтапной фиксации производства работ с автоматическим присвоением номеров и отметками времени;
- гибкого поиска и выборки нарядов по объектам, ответственным лицам, членам бригады и содержанию работ.

В результате обеспечена централизованная система учёта нарядов-допусков, минимизированы риски, связанные с человеческим фактором, а процессы оформления приведены в соответствие с требованиями охраны труда.
